# *Do Shell Script para o Java (Reescrita Completa)*

#### TextAdventure: Projeto de Portabilidade Didática
Este repositório contém um jogo de aventura em formato de texto (Text Adventure) desenvolvido em **Java**. Para rodar de forma pura diretamente na interface de linha de comando (CLI), valorizando a experiência clássica de jogos de terminal.

---
# A Origem em Cli: 
O jogo foi inteiramente concebido, estruturado e testado primeiro em **Shell Script (Bash)**, rodando de forma procedural no terminal Linux.


---
# Adaptação para Java:
Foi todo **refeito do zero em java** mantendo as regras e lógica.
